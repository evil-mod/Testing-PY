# AetherMon

A Pokemon-style creature battle RPG built in Python. Pick a starter, battle wild creatures, collect your team, and explore three routes — all in your terminal.

Features creature classes (Warrior, Mage, Rogue, Priest, Ranger), physical moves and MP-based spells, six status effects, EXP levelling, a town with a shop, and three display modes: plain text, ASCII art, or full-color block sprites.

---

## Prerequisites

- Python 3.10 or newer
- pip

---

## Setup

```bash
# 1. Create and activate a virtual environment
python -m venv .venv
.venv\Scripts\activate       # Windows
source .venv/bin/activate    # Mac / Linux

# 2. Install dependencies
pip install -r requirements.txt

# 3. Generate sprite assets (optional — needed for block mode)
python scripts/download_assets.py
```

---

## Running the Game

```bash
python main.py                 # ASCII art mode (default)
python main.py --mode text     # Plain text only
python main.py --mode ascii    # ASCII art
python main.py --mode block    # Full-color block sprites (requires assets)
```

---

## Tests

```bash
# Unit tests
python -m pytest tests/ -v

# Full automated play-through (covers all menus, all battle actions, all routes)
python -X utf8 scripts/autoplay_test.py
```

## Project Structure

```

├── main.py                  # Entry point
├── requirements.txt
├── assets/sprites/          # PNG sprites (generated by scripts/download_assets.py)
├── scripts/
│   ├── download_assets.py   # Generates CC0 placeholder sprites
│   └── autoplay_test.py     # Automated QA play-through
├── src/
│   ├── data/                # Type chart, moves, classes, creature definitions
│   ├── entities/            # Creature and Player classes
│   ├── engine/              # Battle logic, game loop, display
│   └── ui/                  # Terminal renderer (Rich), ASCII art
└── tests/
    ├── test_battle.py
    └── test_creatures.py
```
