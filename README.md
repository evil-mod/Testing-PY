# evil-mod / Testing

A collection of sample projects used for learning, experimentation, and testing Python concepts.

This repo is part of an ongoing project where we work through coding ideas — games, tools, and small apps — as a hands-on way to explore Python programming. It's a working space, not a polished product repo: expect in-progress builds, experiments, and kids-friendly sample code alongside more serious test work.

---

## Projects

### `games/AetherMon`

A Pokemon-style turn-based RPG built in Python. Kids and beginners learn game logic concepts through a real, playable project:

- Creature collection, levelling, and type matchups
- Physical moves and MP-based spells
- Wild encounters, a rival battle, town shops, and three explorable routes
- Three terminal display modes: text, ASCII art, block sprites
- Full unit test suite + automated play-through QA script

**Python 3.10+** required. See [`games/AetherMon/README.md`](games/AetherMon/README.md) for setup and run instructions.
